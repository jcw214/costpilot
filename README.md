# CostPilot

> 프로젝트 기반 서비스 기업을 위한 원가/관리회계 통합 대시보드 시스템

인력, 프로젝트, 본부, 전사 단위의 **원가 집계**, **내부대체가액 산정**, **표준원가 배분**, **원가/성과 요인 분석**을 수행합니다.

## 주요 기능

| 모듈 | 설명 |
|---|---|
| **대시보드** | KPI 4종(매출·총원가·이익률·가동률) + 본부별 공헌이익 차트 |
| **원가 집계** | 전사→본부→프로젝트→인력 4단계 Drill-Down 원가 조회 |
| **내부대체가액** | 가격정책 × 배부기준 조합별 간접비 배분 시뮬레이션 |
| **표준원가 배분** | 직급별 표준시급·유형별 표준공수 기반 기대 원가 산출 및 실제 비교 |
| **원가 요인 분석** | 직접노무비(임률·능률) / 간접원가(예산·능률·조업도) 3분법 차이 분석 |
| **성과 요인 분석** | 본부 수익성, 프로젝트 이익률, 인력 가동률 종합 평가 |
| **기준 데이터** | 마스터 데이터 조회 및 핵심 필드(시급, 계약매출) 인라인 수정 |
| **거래 데이터** | 투입 공수·직접경비·간접경비 거래 내역 조회 |
| **설정** | 표준원가 기준표 참조, 가격 정책 배율 관리, 배부 기준 조회 |

## 기술 스택

| 계층 | 기술 |
|---|---|
| Frontend | Next.js 14 (TypeScript) + Recharts |
| Backend | Spring Boot 3.4 (Java 21) + JPA |
| Database | PostgreSQL 17 |
| Infra | Docker Compose, GitHub Actions CI/CD |

## 프로젝트 구조

```
costpilot/
├── backend/                        # Spring Boot API 서버
│   └── src/main/java/com/costpilot/
│       ├── organization/           # 본부·인력·프로젝트·직급
│       ├── analysis/               # 원가 집계·내부대체·표준원가·차이 분석
│       ├── budget/                 # 표준원가율 관리
│       ├── dashboard/              # 대시보드 KPI·성과 분석
│       └── global/                 # 예외 처리, 보안, 초기 데이터
├── frontend/                       # Next.js 대시보드
│   └── src/
│       ├── app/                    # 9개 페이지 (App Router)
│       ├── components/             # 공통 UI (SortableTable, HelpPanel 등)
│       ├── features/               # 도메인별 기능 모듈
│       └── lib/                    # API, 포맷, 상수, 도움말 콘텐츠
└── .agents/rules/                  # 설계 문서 (6개)
```
